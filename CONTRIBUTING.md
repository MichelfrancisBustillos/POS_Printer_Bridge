## How to Contribute

1. **Fork the Repository**: Start by forking the repository to your own GitHub account.
2. **Create a New Branch**: Create a new branch for your feature or bug fix.
3. **Make Your Changes**: Implement your changes in the new branch.
4. **Test Your Changes**: Ensure that your changes work as expected and do not break existing functionality.
5. **Submit a Pull Request**: Once you are satisfied with your changes, submit a pull request to the main repository.

## Contribution Guidelines

1. **Code Style**: Follow the existing code style and conventions used in the project.
2. **Commit Messages**: Write clear and descriptive commit messages.
3. **Testing**: Include tests for new features and ensure existing tests pass.
4. **Documentation**: Update documentation as needed to reflect your changes.
5. **Respect the Project Maintainers**: Be courteous and respectful in your interactions with project maintainers and other contributors.
6. **Issue Tracking**: If you are fixing a bug, please reference the issue number in your pull request.
7. **Licensing**: By contributing to this project, you agree that your contributions will be licensed under the same license as the project.

## Reporting Issues

If you encounter any issues or bugs while using the POS Printer Bridge, please report them by creating a new issue in the GitHub repository. Provide as much detail as possible, including steps to reproduce the issue, your environment, and any relevant logs or error messages.

## Development Environment

To set up a development environment for contributing to the POS Printer Bridge, follow these steps:

1. Clone the repository to your local machine.

   `git clone https://github.com/MichelfrancisBustillos/POS_Printer_Bridge`

2. Install the required dependencies using the provided `requirements.txt` file.

    `pip install -r requirements.txt`

3. Run the application locally to test your changes.

   `python src/main.py`

4. Use the provided Docker setup for testing in a containerized environment.

    `docker compose up --build -d`

Thank you for helping to improve the POS Printer Bridge! Your contributions are greatly appreciated.
